二、模板语法

  1. 双大括号插值
  2. v-text——元素的innerText属性；
  3. v-html——元素的innerHtml；
  4. v-bind——绑定属性；（:）
  5. v-if——判断是否插入这个元素，相当于元素的销毁和创建；
  6. v-for——循环输出；
  7. v-on——绑定事件；（@）
  8. v-show——隐藏元素，会给元素的style加上display:none;
  9. v-model——双向数据绑定；
